# Zullkit Frontend Vuejs 3 x BWA Bootcamp

Tech Stack :
- Vue js 3
- Pinia
- Tailwind
- Node js
- Github
- Git
- Deploy with Netlify

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
